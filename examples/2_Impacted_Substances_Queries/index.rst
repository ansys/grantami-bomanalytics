Impacted substances examples
============================

These examples show how to perform an impacted substance query on materials, parts, or XML BoMs in a Granta MI
Restricted Substances database, identifying any substances impacted by one or more legislations. You can write an
equivalent script for specifications. For more information, see :ref:`ref_grantami_bomanalytics_api_index`.
