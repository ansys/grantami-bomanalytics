Impacted Substances Examples
============================

These scripts demonstrate how to determine the substances impacted by one or more legislations for materials and parts
in a Granta MI Restricted Substances database. An equivalent script can be written for Specifications, consult the
:ref:`ref_grantami_bomanalytics_api_index` for details. To determine the substances impacted by an XML BoM, see
:ref:`ref_grantami_bomanalytics_advanced`.
